## Node JS (Express) Boiler Plate
### 1. Clone the repository
``
git clone https://github.com/kiritoriq/node-mongo.git
``
### 2. Copy .env.example and adjust your configuration in .env
``
cp .env.example .env
``
### 3. Run:
``
npm install
``
### 4. Then, run:
``
node index
``
or install nodemon globally, and then run:
``
nodemon index
``
